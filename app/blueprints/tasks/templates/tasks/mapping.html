Mapping 定義說明
欄位順序（建議）
欄位名稱 用途
檔案名稱/資料夾名稱/文字內容來源檔案或資料夾，或在 Add Text / 類型=Add Text 時填入要插入的文字；可填檔名、資料夾名或相對路徑，例如 FolderA/source.docx、FolderA/IFU。若有同名檔案或資料夾，請改填相對路徑
擷取段落/操作章節/指令內容（章節支援連續擷取如 1.1.1-1.1.3、1.1.1-1.1.3 General description、1.1.1 General description - 1.2.2 Intended users、1.1.1 General description - 1.2.2 Intended users\Device trade name；可含 |Figure x. or Table x.、|title=、|index=）；All / PDF Image / 類型=Add Text / 類型=Add Image / 單一 Copy File / 單一 Copy Folder 時直接留白；若要搜尋複製，則填逗號分隔關鍵字搜尋檔名/資料夾名稱
類型空白=擷取章節、All=擷取全文、Figure=擷取圖片、Figure Table=擷取表格中的圖片、Table=擷取表格、PDF Image=將 PDF 每頁轉成圖片插入 DOCX、Add Text=插入文字、Add Image=插入純圖片檔（支援 png/jpg/jpeg/bmp/gif/tif/tiff）、Copy File=複製單一檔案或依檔名搜尋後複製多個檔案、Copy Folder=複製單一資料夾或依名稱搜尋後複製多個資料夾
包含標題控制是否保留標題；章節=章節標題，圖片/表格=caption/title。留白預設包含，可填 是/否、Y/N、TRUE/FALSE
檔案路徑輸出資料夾（相對路徑）
檔案名稱輸出檔名（.docx）；Copy File / Copy Folder 可留白，留白時沿用來源名稱。若發生同名衝突，系統會自動補如 _knee / _hip 後綴
模板文件可空白；需插入模板段落時填入
插入段落名稱/目的資料夾名稱模板模式下的插入段落名稱
操作欄語法範例
若需隱藏標題，請在 包含標題 欄位填入 否 或 FALSE。
頁面上的 檢查引用文件 會驗證檔案、模板與欄位格式；檢查擷取參數 會額外實際驗證章節、圖片、表格是否可被擷取。
若章節標題本身包含 / 或 \，請用雙引號包住標題，例如 2.1.6 "Instructions for use/Device Operating Manual(s)"。
章節擷取範例1（類型 空白）：1.1.1 General description
章節連續擷取範例2（類型 空白）：1.1.1 General description - 1.2.2 Intended users
章節連續擷取範例3（類型 空白）：1.1.1 General description - 1.2.2 Intended users\Device trade name
章節連續擷取範例4（相容舊寫法）：1.1.1-1.1.3 General description
擷取全文範例（類型 All）：操作欄留白
表格擷取範例2（類型 Table）：1.1.1 General description|title=Table summary|Figure 1.
表格擷取範例3（類型 Table）：1.1.1 General description|title=Table summary|index=1
表格擷取範例4（類型 Table）：1.1.1 General description|title=Table summary|title=table title
圖片擷取範例1（類型 Figure）：1.1.1 General description|title=System overview|Figure 2.
圖片擷取範例2（類型 Figure）：1.1.1 General description|title=System overview|index=2
圖片擷取範例3（類型 Figure）：1.1.1 General description|title=System overview|title=figure title
圖片擷取範例4（表格包圖，類型 Figure Table）：Packaging for the Implant|Figure 2
插入文字範例（類型 Add Text）：操作欄留白 或 Add Text
插入圖片範例（類型 Add Image）：操作欄留白 或 Add Image
PDF 圖片擷取範例（類型 PDF Image）：操作欄留白
複製檔案範例1（類型 Copy File，單一檔案）：操作欄留白
複製檔案範例2（類型 Copy File，依檔名搜尋）：IFU 或 shipping,EO
複製資料夾範例1（類型 Copy Folder，整包複製）：操作欄留白
複製資料夾範例2（類型 Copy Folder，依名稱搜尋）：IFU 或 Label,sample
