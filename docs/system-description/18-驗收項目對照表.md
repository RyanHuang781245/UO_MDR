## 對照原則

本表依使用者提供之驗收項目整理。若項目可由目前程式碼或文件確認，列出對應章節、主要程式與說明；若無法確認完整設計或驗收範圍，標示「此處需補充」。

| 驗收項目 | 對應章節 | 主要程式或資料來源 | 說明 | 待補充 |
|---|---|---|---|---|
| 核心資料程式 | 第 1、3、9、15、16 章 | `app/models/task.py`、`app/models/execution.py`、`app/models/mapping_metadata.py`、`app/models/auth.py`、`app/models/settings.py` | 系統核心資料包含任務、工作、Mapping metadata、使用者角色、稽核、系統錯誤與系統設定。 | 此處需補充驗收方所稱「核心資料」是否包含標準更新資料表。 |
| 任務管理與檔案匯入程式 | 第 3、4、5 章 | `app/blueprints/tasks/task_routes.py`、`app/blueprints/tasks/nas_routes.py`、`app/services/task_service.py`、`app/services/nas_service.py` | 任務建立、任務列表、NAS 來源解析、背景同步、任務檔案管理與任務 metadata。 | 此處需補充正式 NAS root 與來源同步 worker 的完整複製規則。 |
| 流程管理與步驟程式 | 第 6、7、10、15 章 | `app/blueprints/flows/execution_routes.py`、`app/blueprints/flows/flow_crud_routes.py`、`app/services/flow_definition_service.py`、`app/services/flow_validation_service.py`、`modules/workflow.py` | 流程 JSON 建立、保存、匯入、匯出、執行前驗證與工作步驟執行。 | 此處需補充流程 JSON schema 的正式規格。 |

| 其它相關程式 | 第 14、15、16 章 | `app/services/auth_*`、`app/services/standard_update_service.py`、`app/services/operations_service.py`、`app/jobs/adoption_standard_update.py` | 包含權限、管理後台、標準更新、維運 CLI、備份/清理文件與部署支援。 | 此處需補充驗收方所稱「其它」的範圍。 |
