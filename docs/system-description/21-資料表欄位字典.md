## 說明

以下欄位由 SQLAlchemy metadata 產生。型別以 SQLAlchemy 測試設定輸出的型別表示，正式 MSSQL 實體型別會由 SQLAlchemy dialect 轉換。`PK` 表示主鍵，`Nullable` 表示是否允許空值。
